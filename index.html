Password Checker Project

This project is designed to securely check if any of your passwords have ever been hacked, using a Password API, Hasing Generator, and K-Anonymity. This Project allows anyone the visibility to know how secure their passwords, without compromising said password. One of the challenges I encountered in this project was really making sure that security was front of mind when dealing with anyone's personal information. I believe we overcame that with the SHA-1 Hashing Generator, and ensuring that no individuals can be identified. I hope you enjoy reviewing this project as much as I enjoyed building it!
